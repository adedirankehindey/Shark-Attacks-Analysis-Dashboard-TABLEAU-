# 🚀Project Overview
This project analyzes historical shark attack data to understand patterns in attacks across geography, species, injury severity, gender distribution, and time trends.

The dashboard transforms raw shark incident data into a visual analytical report that helps users quickly identify risk patterns, attack trends, and key environmental insights.

---

# 🎯Problem Statement
Shark attacks occur globally but are often poorly understood due to scattered reporting and limited visualization of trends.

Key analytical questions addressed include:
- Which shark species are responsible for the highest number of attacks?
- How have shark attacks changed over time?
- Which countries report the most incidents?
- Are attacks more common among certain demographics?
- What types of injuries are most frequent?

Without structured analysis, these patterns remain hidden in raw datasets.

---

# 🗂Data Structure and Overview
The dataset contains historical records of shark attack incidents.

Primary fields include:
- Country
- Species
- Gender
- Age
- Year of incident
- Type of shark attack
- Injury description
- Location

Data was stored in CSV format and imported into Tableau for analysis and visualization.

Key dataset characteristics:
- Total attacks analyzed: 1,210
- Countries represented: 55
- Attack locations recorded: 969
- Average victim age: 29 years

# 🔎Data Analysis Process

Step 1: Data Collection  
Historical shark attack data was imported from CSV files.

Step 2: Data Cleaning  
Missing values, inconsistent species naming, and formatting issues were corrected.

Step 3: Exploratory Data Analysis  
Basic frequency analysis was performed to identify patterns in:
- species distribution
- gender
- geographic location
- injury types

Step 4: Dashboard Design  
Multiple visualizations were created in Tableau including:
- species distribution bar chart
- gender comparison chart
- injury category analysis
- yearly attack trend line

Step 5: Interactive Filtering  
Filters were implemented for:
- country
- gender
- shark species
- attack type

---

# 💼Executive Summary
The dashboard reveals several key insights, which includes:

- White sharks account for the largest share of recorded attacks.
- Male victims represent the majority of reported incidents.
- Shark attacks have increased in reporting frequency in recent decades.
- Certain countries show higher incident concentration due to coastal activity and tourism.
- Injury severity varies widely, with leg and foot injuries being common.

---

# 💡Recommendation
Based on the analysis:

- Coastal safety authorities should monitor high‑risk shark species regions.
- Public education programs should target high‑exposure groups such as surfers and divers.
- Tourism authorities should implement real‑time shark monitoring systems.
